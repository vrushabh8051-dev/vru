Resume & Biodata Site for Vrushab

- index.html — Resume overview
- biodata.html — Detailed biodata using semantic elements
- style.css — Styles
- main.js — jQuery interactions

To view locally:
1. Open `c:/xampp/htdocs/vrush/index.html` in a browser, or
2. If using XAMPP, place folder under `htdocs` and visit `http://localhost/vrush/`.

This is a simple static site — no build step required.
